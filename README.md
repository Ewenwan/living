## G. 个人形象
### 伴郎需要做什么
    他们“静”可以涂口红，“动”可以尬舞、跳绳，文能做智力脑洞题，
    武能用指压板变着法折磨自己……接亲游戏没有一个难得到他们的，分分钟完美搞定！
    
    
1、伴郎是一个婚礼工作组中的重要人物，婚礼当天，伴郎除了服侍新郎、处理一切临时突发状况外，还要带领宾客一起玩新人。

2、在迎娶时，先替新郎和这群伴娘们交战，以帮助新郎顺利取得美人归。

3、除了随时替新郎注意仪容，安抚新郎的心情，替新郎拿东西、开车门外，伴郎有一个主要功能，就是要在会场门口担任招待的工作，引导来宾顺利入室。

4、伴郎的另一个功能便是要替新郎挡酒，在新郎浅尝表意之后，都要快步当前，代新郎先饮为敬。

5、伴郎还有一个最主要的任务就是带领宾客一起玩新人，如果宾客玩游戏不起来，伴郎一定要先行开场主持，还要小心注意尺度的问题，千万不要因为好玩，而把新娘惹怒了。

婚礼仪式前

    1、准备工作：婚礼当天伴郎一定要确保新郎准时起床并穿戴整齐，检查有没有遗忘的物品。
    2、接亲仪式：迎娶时伴郎要帮助新郎解决伴娘的“刁难”，进入闺房，顺利接亲。
    3、外景拍摄：婚礼当天一般都会有外景拍摄，这时伴郎也注意准备好足够的食物和饮料以及拍摄道具，以备不时之需。
    
婚礼仪式中

    1、迎接宾客：伴郎应该要陪同新人一起迎接和感谢来宾，帮助引导来宾顺利入座。
    2、婚礼仪式：在婚礼仪式中，有些时候伴郎负责保管新人的戒指，并将其适时地交给新郎。
    3、敬酒程序：伴郎时常需要替新人挡酒，并做好与客人的交流工作，这步相信大部分伴郎都知道。
    4、换衣过程：婚礼当天新人一般都需要换服装，伴郎一定要帮助新郎检查一下衣着有没有问题。
    5、其他职责：伴郎还需要在婚礼上发放答谢来宾的喜糖和礼物，如果婚礼上有比较特别的节目，
       伴郎还需要帮助调动婚礼的气氛。
    
婚礼仪式后

    1、闹洞房：婚礼仪式结束后，伴郎还要带领宾客一起热热闹闹地闹洞房。

    2、送宾客，收拾场地：伴郎可不能先走，一定要把最终的收尾工作做好。
       比如帮助新人收拾一下闹洞房时造成的狼藉场景，
       跟新人汇报一下摆放礼品的地方及归还新人交托的各类物品等等。
       
婚礼结束

    另外伴郎要时刻关注婚礼现场的各种状况，及时将小状况都解决好。
    伴郎一定要随时保持笑容，灵活待客。
    
    最好要准备一些祝酒词，
    比如介绍新郎在单身的时候是个怎样的人，或者开些玩笑，最后再来表示对新人的祝福。
    
    在跟来宾交流的时候可以适时地开一些玩笑，这样可以活跃婚礼现场的氛围。
    婚礼仪式正式举行的时候，伴郎要与伴娘一起，等候在舞台的一侧，随时准备为新人服务。
    希望即将要当伴郎的你能做一个合格的伴郎。
    
    
    一般婚礼结束，一般都会有小型的party，此时伴郎团的工作已经接近尾声，
    是时候放飞自我，展示你们真正的才艺啦！吹拉弹唱的载歌载舞，捧哏逗眼的愉悦宾客，都要露一手。
